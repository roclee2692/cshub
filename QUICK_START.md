## 🛠️ 技术实现（开发者指南）

### 为新算法添加代码同步支持

#### 方式 1：最简单（推荐）
```javascript
// 在算法步骤中直接添加行号
steps.push({
  // ... 其他字段
  cppLine: 6,     // C++ 代码的行号
  pythonLine: 6,  // Python 代码的行号
  description: '...',
})
```

#### 方式 2：使用对象映射
```javascript
steps.push({
  // ... 其他字段
  codeLines: {
    cpp: 6,
    python: 6,
    java: 6,  // 未来扩展
  },
  description: '...',
})
```

### 验证代码行号

确保：
1. 行号是 **1-indexed**（第一行是 1，不是 0）
2. 行号对应代码块的 **实际行数**
3. 可选：添加注释说明映射关系

```javascript
// C++ code line mapping (1-indexed):
// Line 3: for (int i = 0; i < n - 1; i++)
// Line 4: bool swapped = false;
// Line 5: for (int j = 0; j < n - i - 1; j++)
// ...
```

## ❓ 常见问题

### Q: 为什么某个算法没有显示代码？
**A:** 两个可能的原因：
1. 这个算法的可视化类型（如 graph、tree）不适合并排显示
2. 这个算法还没有添加代码行号映射

### Q: 代码行号为什么不高亮？
**A:** 检查：
1. 是否在步骤对象中添加了 `cppLine`/`pythonLine` 字段？
2. 行号是否正确（应对应代码块的实际行）？
3. 浏览器控制台是否有错误信息？

### Q: 窄屏上代码怎么看？
**A:** 三个选项：
1. 自动竖排显示（上方动画，下方代码）
2. 点击 "⬅️ 并排" 回到并排显示（可能有重叠）
3. 放大浏览器或使用更大的设备

### Q: 能支持其他编程语言吗？
**A:** 可以！当前支持 C++ 和 Python，未来可以添加：
- Java、Go、Rust 等
- 只需在代码映射中添加新语言即可
