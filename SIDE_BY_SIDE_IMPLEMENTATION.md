# 并排显示动画和代码 - 实现完成

## 🎯 核心改进

### 问题解决
- ✅ 伪代码不再跟动画同步，保持静态显示
- ✅ 焦点不被转移，动画清晰可见  
- ✅ **新增：动画和代码并排显示**
- ✅ C++/Python 代码与动画实时同步

### 新增功能

#### 1. InteractiveVisualization 组件
- 位置：`src/components/learning/InteractiveVisualization.jsx`
- 功能：并排显示动画和代码
- 特性：
  - 自动从步骤数据中提取 `cppLine`/`pythonLine`
  - 实时高亮对应代码行
  - 支持 C++/Python 语言切换
  - 响应式设计（屏幕宽度 < 1400px 时自动竖排）
  - 手动布局切换按钮（⬇️ 竖排 / ⬅️ 并排）
  - Sticky 定位（代码块跟随滚动）

#### 2. 改进的页面布局
- 位置：`src/pages/AlgorithmPage.jsx`
- 变化：
  - 导入 `InteractiveVisualization` 组件
  - "交互式可视化" 部分现在使用并排布局
  - 自动检测可视化类型（仅排序算法显示代码）
  - 定义了 `VIZ_WITH_CODE` 集合来标记哪些类型支持并排代码

#### 3. 算法支持代码行号映射

已实现的算法：
- ✅ **bubbleSort** - 冒泡排序
- ✅ **insertionSort** - 插入排序
- ✅ **selectionSort** - 选择排序

计划实现：
- ⏳ heapSort - 堆排序
- ⏳ countingSort - 计数排序
- ⏳ shellSort - 希尔排序

复杂结构（需要特殊处理）：
- ❓ quickSort - 快速排序（自定义数据结构）

## 💻 使用指南

### 对于学习者
1. 打开支持并排显示的算法页面（如冒泡排序）
2. 左侧看动画，右侧看代码
3. 点击播放按钮，代码行自动高亮
4. 点击 "C++" / "Python" 标签切换代码语言
5. 点击 "⬇️ 竖排" 按钮可以切换布局方式

### 对于开发者：为算法添加代码行号

#### 步骤 1：在算法文件顶部添加映射注释
```javascript
// C++ code line mapping (1-indexed):
// Line 3: for (int i = 0; i < n - 1; i++)
// Line 4: bool swapped = false;
// ...

// Python code line mapping (1-indexed):
// Line 3: for i in range(n - 1):
// Line 4: swapped = False
// ...
```

#### 步骤 2：在步骤对象中添加代码行号
```javascript
steps.push({
  array: [...arr],
  comparing: [j, j + 1],
  swapped: [],
  sorted: [...sortedIndices],
  cppLine: 6,        // ← 添加
  pythonLine: 6,     // ← 添加
  description: `比较 arr[${j}]=${arr[j]} 与 arr[${j+1}]=${arr[j+1]}`,
})
```

#### 步骤 3：测试
- 打开算法页面
- 验证代码行是否正确高亮
- 切换语言标签检查两种语言的高亮

## 📊 支持代码并排显示的算法类型

当前配置在 `AlgorithmPage.jsx` 中：
```javascript
const VIZ_WITH_CODE = new Set(['sorting', 'heap', 'counting'])
```

**sorting** - 基础排序算法
- 冒泡排序 ✅

- 选择排序 ✅

- 计数排序 ⏳

- 归并排序 ❓
- 堆排序 ⏳

**heap** - 堆排序
- 可视化：堆的插入/删除
- 代码支持：⏳

**counting** - 计数排序
- 可视化：直方图
- 代码支持：⏳
