# 可隐藏侧栏 - 快速使用指南

## 🎯 功能说明

顶部栏新增了一个**侧栏折叠按钮**，可以快速隐藏/展开左侧导航栏，为动画和代码显示腾出更多空间。

## 📍 按钮位置

```
┌─────────────────────────────────────────────────────┐
│ [≡ 折叠]  AlgoViz  [首页] [算法库]   [搜索]  [主题]  [GitHub] │
│  ↑
│  新增的折叠按钮（仅桌面版）
└─────────────────────────────────────────────────────┘
```

## 🎮 使用方法

### 隐藏侧栏
1. 点击顶部栏左侧的**折叠按钮** （三条竖线图标）
2. 侧栏平滑滑出，动画和代码区域自动扩展

### 展开侧栏
1. 点击相同位置的**展开按钮** （反向三条竖线图标）
2. 侧栏平滑滑入

### Tooltip
- 悬停在按钮上会显示提示文字：
  - "隐藏侧栏" / "展开侧栏"

| 状态 | 屏幕宽度 | 侧栏宽度 | 主区域宽度 | 动画/代码 |
|------|---------|---------|----------|---------|
| **展开** | ≥ 1648px | 248px | 1400px | **并排** |
| **展开** | < 1648px | 248px | 减少 | **竖排** |
| **隐藏** | ≥ 1348px | 0px | 1100px+ | **并排** |
| **隐藏** | < 1348px | 0px | 全屏 | **竖排** |

## 🖥️ 设备支持

### ✅ 支持
- 桌面浏览器（Windows, Mac, Linux）
- 屏幕宽度 ≥ 768px 的平板设备
- Chrome, Firefox, Safari, Edge 最新版本

### ⚠️ 不支持
- 手机屏幕（< 768px）

  - 点击菜单按钮展开/隐藏侧栏

- **图标**：SVG 矢量图标，自动适应亮/暗主题

## 🐛 常见问题

### Q: 为什么侧栏没有立即消失？
A: 侧栏有 300ms 的平滑过渡动画，这是设计特性，提升视觉体验。

### Q: 我的手机上没有看到折叠按钮？
A: 这是正常的。手机屏幕使用侧栏抽屉模式（点击菜单按钮展开/隐藏），不需要折叠按钮。

### Q: 侧栏隐藏后为什么动画还是竖排显示？
A: 因为你的屏幕宽度不足 1348px。需要在屏幕宽度 ≥ 1348px 时才能并排显示。

### Q: 刷新页面后侧栏状态会重置吗？
A: 是的，当前版本刷新后会回到展开状态。可以通过 localStorage 实现记忆（见高级配置）。

## ⚙️ 高级配置

### 保存侧栏状态（可选）

如想让浏览器记住你的偏好，可修改 `AppLayout.jsx`：

```jsx
// 初始化时从 localStorage 读取
const [sidebarCollapsed, setSidebarCollapsed] = useState(() => {
  if (typeof window !== 'undefined') {
    return localStorage.getItem('sidebarCollapsed') === 'true'
  }
  return false
})

// 状态变化时保存
useEffect(() => {
  localStorage.setItem('sidebarCollapsed', sidebarCollapsed)
}, [sidebarCollapsed])
```

### 修改过渡时间

在 `AppLayout.jsx` 中修改 transition 属性：

```jsx
<div style={{
  transition: 'all 0.3s ease',  // ← 改这里
  width: sidebarCollapsed ? 0 : 248,
}}>
```

- `0.3s` → `0.1s`：更快
- `0.3s` → `0.5s`：更慢

## 🎯 使用场景

### 场景 1：观看动画讲解

4. **宽屏显示** → 随意切换，无性能影响
