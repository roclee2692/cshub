# 可调节分割面板功能使用指南

## 📱 iPad 风格分屏功能

算法可视化工具现在支持 iPad 风格的可调节分割面板，让你能够灵活调整动画窗口和代码窗口的大小。

## 🎯 功能特性

### 1. **自由拖动调整大小**
- 在动画和代码之间的分割线上拖动，可以实时调整两个窗口的宽度
- 分割线会在悬停时变为高亮，提示可以拖动
- 拖动时分割线会闪烁表示正在调整

### 2. **智能响应式设计**
- **宽屏显示**：默认并排显示动画和代码
- **窄屏自动调整**：在小屏幕上自动切换为竖排显示
- **用户偏好记忆**：分割比例保存在 localStorage，刷新后自动恢复

### 3. **最小宽度保护**
- 每个面板都有最小宽度限制（250px），防止任一面板过小而无法使用
- 自动限制拖动范围，确保两个窗口都保持可用

## 🖱️ 使用方法

### 基本操作
1. **打开算法页面**：点击任意算法进入学习页面
2. **查看分割面板**：在宽屏上会看到左侧动画和右侧代码
3. **拖动调整**：将鼠标移至中间分割线，光标变为 `↔️` 时拖动即可调整大小

### 布局切换
- **并排显示**：在宽屏上默认并排显示
- **竖排显示**：点击右上方的 **⬇️ 竖排** 按钮手动切换
- **切回并排**：点击 **⬅️ 并排** 按钮切回

### 代码语言切换
- 点击 **C++** 或 **Python** 标签切换代码语言
- 动画步骤会自动高亮对应的代码行

## 💾 数据保存

- 分割比例会自动保存到浏览器的 localStorage
- 每个算法有独立的比例设置（按算法 slug 区分）
- 刷新页面或关闭后重新打开，分割比例会自动恢复

## 🎓 学习场景

### 适合场景
- **理解算法逻辑**：同时看动画执行过程和对应代码
- **代码学习**：追踪每一步的代码执行
- **教学演示**：讲师可以调整分割比例突出重点
- **对比学习**：在对比页面（/compare）并排对比两个算法的实现

### 使用建议
1. **初学者**：建议动画窗口稍大些（60% : 40%），重点看动画
2. **深度学习**：建议代码窗口稍大些（40% : 60%），深入理解代码
3. **教学**：建议 50:50 分割，方便学生同时看动画和代码

## 🔧 技术细节

### 组件结构
```
InteractiveVisualization
└── ResizableSplitPanel
    ├── 左面板（动画可视化）
    ├── 分割线（可拖动）
    └── 右面板（代码展示）
```

### localStorage 键名
```
split-ratio-{算法slug}
// 例如：split-ratio-bubblesort
```

### 响应式断点
- **宽屏**（> 1100px）：并排显示 + 可调节
- **中屏**（800-1100px）：竖排显示（可手动切换）
- **小屏**（< 800px）：竖排显示（强制）

## 🐛 故障排除

### 分割线不见了
- 尝试刷新页面
- 检查浏览器开发者工具是否有 JavaScript 错误

### 分割比例没有保存
- 检查浏览器是否允许 localStorage
- 清除浏览器缓存后重试

### 分割线拖动卡顿
- 这是正常的（取决于浏览器和硬件），刷新页面重试
- 尝试关闭浏览器扩展

## 📊 对比页面

在 `/compare` 页面，每个算法的比较面板也支持同样的可调节功能，让你能够：
- 并排看两个算法的动画
- 同时查看两种算法的代码
- 逐步调整面板大小以突出你感兴趣的部分
